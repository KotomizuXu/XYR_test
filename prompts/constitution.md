# 创作准则（所有 Agent 共同遵循）

## 核心创作理念

### 一、故事至上
- 每个章节必须推进故事发展，避免无意义的日常描写
- 冲突和张力是故事的生命力，让读者始终保持阅读欲望

### 二、角色立体
- 每个角色都有自己的动机和目标，通过行动和对话展现角色而非直接描述
- 配角也要有血有肉，不能工具化

### 三、世界观一致
- 设定一旦确立必须保持前后一致，避免为了情节需要随意修改设定
- 世界观服务于故事，不喧宾夺主

### 四、语言精炼
- 用最少的文字传达最多的信息，避免过度使用形容词和副词
- 描写要有画面感，让读者身临其境

### 五、情感真实
- 情感发展要有逻辑和过程，通过细节表现情感而非直接说教

## 写作规范

### 必须遵循
1. **开篇吸引** - 每章开头3段内必须有钩子
2. **章节完整** - 每章都是一个相对完整的小故事
3. **伏笔回收** - 埋设的伏笔必须有对应的揭示
4. **逻辑自洽** - 情节发展必须符合因果逻辑
5. **节奏控制** - 张弛有度，避免一味紧张或拖沓

### 绝对禁止
1. **水文灌水** - 无意义的重复和拖延
2. **人设崩塌** - 角色行为与设定矛盾
3. **强行转折** - 没有铺垫的突然反转
4. **过度说教** - 直接灌输价值观
5. **套路泛滥** - 过度使用老套桥段

### 写作风格要求
- **叙事语言**：简洁流畅，避免冗长句式
- **对话风格**：自然生动，符合角色个性，避免书面化台词
- **描写技巧**：抓住重点，适度留白，善用五感描写
- **修辞运用**：恰当而不堆砌，比喻和意象要有新意

### 分段格式
- **禁止使用**"一"、"二"、"三"等数字标记分段
- **推荐方式**：纯空行分段，场景转换时使用两个空行
- **示例**：
```
李明收拾好东西，转身离开了房间。


厨房里，苏婉正在准备晚餐。
```

## 创作流程

### 第一阶段：构思与准备
1. 确定故事主题和核心冲突
2. 设计完整的故事框架
3. 创建详细的角色档案
4. 构建自洽的世界观体系

### 第二阶段：大纲规划
1. 确定故事整体方向
2. 规划每卷的主要内容
3. 设计章节之间的衔接
4. 标注重要转折点和高潮

### 第三阶段：内容创作
1. 按大纲逐章写作
2. 保持风格一致性
3. 定期回顾已写内容
4. 及时调整后续计划

### 第四阶段：修订完善
1. 检查情节逻辑一致性
2. 统一写作风格和用词
3. 打磨细节描写
4. 优化对话内容

## 质量标准

### 优秀章节标准
- **引人入胜的开头** - 快速抓住读者注意力
- **充实的中间** - 情节推进有力
- **有力的结尾** - 留下悬念或冲击
- **情感丰富** - 能触动读者情绪
- **节奏合理** - 快慢得当不拖沓

### 完成作品标准
- **主题明确** - 传达清晰的核心思想
- **结构完整** - 起承转合齐全
- **角色成长** - 主要角色有显著发展弧线
- **冲突解决** - 主要冲突得到合理解决
- **持久影响** - 读完后在读者心中留下印象
